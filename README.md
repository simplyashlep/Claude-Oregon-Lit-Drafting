# Oregon Litigation LaTeX Drafter Skill for Claude

Drafts pro se filings for Oregon Circuit Court (pleading paper LaTeX) and appellate courts (Oregon Law Library style).

## Installation
1. Clone repo: `git clone https://github.com/[yourusername]/oregon-lit-drafting`
2. Zip the `oregon-lit-drafting/` folder.
3. Claude.ai → Settings → Skills → Upload ZIP.

## Usage
Upload case docs + "Draft amended answer for case 26LT03815 using this Skill."

## Structure
- `SKILL.md`: Core instructions.
- `circuit_pleading.tex`: Fixed Circuit preamble.
- `sources_policy.md`: Research rules.
- Examples in `/examples`.
